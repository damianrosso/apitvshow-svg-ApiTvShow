Política de Privacidad

Última actualización: 30 de marzo de 2026

Esta Política de Privacidad describe cómo ApiTvShow recopila, usa, almacena y protege la información en relación con nuestro servicio de automatización de contenido de entretenimiento y su integración con la plataforma TikTok.

1. Modelo de Uso

ApiTvShow opera bajo un modelo de creador único. La aplicación es utilizada exclusivamente por el propietario de la cuenta de TikTok conectada. No existen cuentas de usuarios externos, páginas de registro, ni interacción de terceros con la plataforma.

2. Información Recopilada

ApiTvShow recopila y almacena la siguiente información de forma local en el dispositivo del creador:

Dato Fuente Propósito
Access Token de TikTok Flujo OAuth 2.0 de TikTok Autenticar y publicar contenido en la cuenta del creador
Refresh Token de TikTok Flujo OAuth 2.0 de TikTok Renovar el access token cuando expira
Open ID del usuario API de TikTok (user.info.basic) Identificar al creador autenticado
Artículos de noticias Sitios de noticias públicos Generar contenido para los videos

3. Uso de la Información

La información recopilada se utiliza estrictamente para:

  • Autenticación: Conectar de forma segura con la API de TikTok para publicar contenido.
  • Publicación de contenido: Subir videos generados automáticamente con subtítulos a la cuenta de TikTok del creador.
  • Refresco de tokens: Mantener activa la sesión de autenticación sin requerir que el creador vuelva a iniciar sesión manualmente.

La información no se usa para:

  • Rastreo, perfilado o publicidad.
  • Analítica o procesamiento de datos de terceros.
  • Venta, cesión o transferencia de datos a ningún tercero.

4. Almacenamiento y Seguridad

Todos los datos se almacenan localmente en el dispositivo del creador en archivos de configuración JSON. No se utilizan servidores en la nube, bases de datos externas ni servicios de almacenamiento de terceros para la información del usuario.

  • Los tokens de acceso se almacenan en archivos de configuración locales protegidos por los permisos del sistema operativo.
  • Los tokens se renuevan automáticamente y los antiguos se sobrescriben.
  • No se transmiten datos personales fuera de las llamadas directas a la API de TikTok.

5. Compartición de Datos con Terceros

ApiTvShow no comparte, vende ni transfiere información personal o tokens de autenticación a ningún tercero. Las únicas comunicaciones externas son:

  • API de TikTok: Para autenticación y publicación de contenido, según lo autorizado explícitamente por el creador.
  • API de Google Gemini: Se envía únicamente el texto del artículo de noticias (sin datos personales) para generar subtítulos.
  • Sitios de noticias públicos: Se accede a contenido de acceso público para recopilar artículos de noticias.

6. APIs de TikTok y Scopes Utilizados

ApiTvShow solicita los siguientes permisos de TikTok, cada uno autorizado explícitamente por el creador durante el flujo de inicio de sesión:

Scope Propósito
user.info.basic Leer información básica del perfil (Open ID, avatar, nombre de usuario) para autenticación.
video.publish Publicar contenido de video directamente en el perfil de TikTok del creador.
video.upload Subir borradores de video a la cuenta del creador para edición y revisión antes de publicar.

7. Retención de Datos

  • Tokens de autenticación: Se conservan mientras la aplicación esté activa. Se sobrescriben en cada refresco.
  • Artículos de noticias: Se conservan por un máximo de 24 horas y luego se eliminan automáticamente.
  • Videos generados: Se almacenan temporalmente en la carpeta local y se sobrescriben en cada nueva ejecución.

8. Derechos del Creador

El creador puede en cualquier momento:

  • Revocar el acceso: Desconectar ApiTvShow desde la configuración de TikTok (Configuración → Seguridad → Gestionar permisos de apps).
  • Eliminar datos locales: Borrar los archivos de configuración de la aplicación que contienen los tokens.
  • Dejar de usar la aplicación: Simplemente dejar de ejecutar ApiTvShow; no quedan datos residuales en la nube.

9. Privacidad de Menores

ApiTvShow no está destinado a ser utilizado por menores de 18 años. La aplicación es operada exclusivamente por el creador propietario de la cuenta y no recopila información de menores.

10. Cambios a esta Política

Esta Política de Privacidad puede ser actualizada periódicamente para reflejar cambios en la aplicación o en los requisitos legales. Los cambios se publicarán en esta página con la fecha de actualización correspondiente.

11. Contacto

Para consultas sobre esta Política de Privacidad o sobre el manejo de tus datos, podés contactarnos a través de nuestro repositorio o por correo electrónico.
