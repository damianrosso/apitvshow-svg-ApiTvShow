ApiTvShow is a desktop content automation platform that curates trending entertainment news from Argentina, generates short-form videos with AI-powered captions, and publishes them directly to the creator's TikTok profile.

What ApiTvShow Does

📰

News Aggregation

Scrapes trending entertainment, TV shows, and celebrity news from verified Argentine media sources including Infobae, TN, and A24.

🤖

AI-Powered Captions

Uses Google Gemini AI to generate engaging, context-aware captions in Spanish optimized for social media engagement.

🎬

Video Generation

Automatically creates short-form vertical videos with text overlays, fade-in animations, and background music using FFmpeg.

📲

Direct TikTok Publishing

Posts generated videos directly to the authenticated creator's TikTok account via the Content Posting API with proper captions and metadata.

📊

Dashboard Management

Web-based dashboard to monitor publishing pipeline, schedule automated posts, review content before publishing, and track quota usage.

🔒

Single-Creator Model

Designed for a single authenticated creator. No external users interact with the app — only the account owner manages and authorizes content.

How It Works

1

Scrape

The platform collects the latest entertainment articles from multiple verified news sources.

2

Enrich

Articles are enriched with metadata, images, and publication dates from the original sources.

3

Generate

AI creates an engaging caption, and FFmpeg composes a branded short-form video with text overlay.

4

Publish

The video is uploaded to the creator's TikTok account via the Content Posting API with the generated caption.

TikTok API Integration

ApiTvShow integrates with TikTok's Content Posting API following all platform guidelines and best practices.

Authentication

Uses OAuth 2.0 with PKCE for secure authorization. The creator explicitly grants access through TikTok's login flow. Tokens are refreshed automatically and stored securely on the local machine.

Content Posting

Videos are uploaded using the FILE_UPLOAD method via the Direct Post endpoint. Each post includes the AI-generated caption as the title and respects the creator's privacy settings.

Scopes Used

user.info.basic — Read profile info for authentication.
video.publish — Post content directly to the creator's profile.
video.upload — Upload video drafts for review before publishing.

User Consent & Data

The creator explicitly authorizes all TikTok interactions through the OAuth flow. No data from external users is collected. The app operates in a single-creator model with no public-facing user accounts.
